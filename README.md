# MAVIS ГРУПП — Анализ звонков 📞

PWA-приложение для анализа качества звонков менеджеров отдела продаж.

## Возможности

- 🔄 Загрузка звонков из Bitrix24 CRM
- 🎙 Автоматическая транскрибация аудио через Claude AI
- 📊 Анализ по 8 критериям с взвешенной оценкой
- 📱 Устанавливается как приложение на телефон/компьютер
- 📈 Отчёты по дням / неделям / месяцам
- 👥 Фильтрация по сотрудникам
- 💾 Сохранение истории анализов

## Быстрый старт — развёртывание на GitHub Pages

### 1. Создайте репозиторий

```bash
# Скачайте ZIP этого проекта или клонируйте
git init mavis-calls
cd mavis-calls
```

### 2. Скопируйте файлы

Положите в корень репозитория все файлы:
```
├── index.html
├── manifest.json
├── sw.js
├── favicon.ico
├── icon.svg
├── icons/
│   ├── icon-72x72.png
│   ├── icon-96x96.png
│   ├── icon-128x128.png
│   ├── icon-144x144.png
│   ├── icon-152x152.png
│   ├── icon-192x192.png
│   ├── icon-384x384.png
│   ├── icon-512x512.png
│   └── apple-touch-icon.png
└── .github/
    └── workflows/
        └── deploy.yml
```

### 3. Запушьте на GitHub

```bash
git add .
git commit -m "Initial deploy"
git remote add origin https://github.com/YOUR_USERNAME/mavis-calls.git
git branch -M main
git push -u origin main
```

### 4. Включите GitHub Pages

1. Откройте репозиторий на GitHub
2. Settings → Pages
3. Source: выберите **GitHub Actions**
4. Подождите 1-2 минуты, деплой произойдёт автоматически

### 5. Готово!

Приложение будет доступно по адресу:
```
https://YOUR_USERNAME.github.io/mavis-calls/
```

## Установка как приложение

### На телефоне (Android / iOS):
1. Откройте ссылку в браузере Chrome / Safari
2. Нажмите «Добавить на главный экран» (или появится баннер)
3. Приложение появится с иконкой MAVIS на рабочем столе

### На компьютере (Chrome / Edge):
1. Откройте ссылку в браузере
2. В адресной строке нажмите иконку установки (⊕)
3. Или: меню → «Установить приложение»
